  \item Realized spread, effective spread, price impact
\end{itemize}

\begin{frame}{Getting the Data from Coincall}
\begin{itemize}
  \item Goal: pull a single option's order book and recent trades, build mid/spread.
  \item Option order book (100 depth):  GET /open/option/order/orderbook/v1/\{symbol\}
  \item Recent trades:                   GET /open/option/trade/lasttrade/v1/\{symbol\}
  \item List symbols first:              GET /open/option/getInstruments/BTC
  \item Response gives arrays of [price, size] for bids and asks -\textgreater{} aggregate (L2)
  \item For replay/backtests use the provided Parquet L2 snapshots (1s for options)
  \item Live updates: subscribe to the /options/orderbook WebSocket channel
\end{itemize}
\vspace{2pt}\tiny\textit{Cross-check exact paths at https://docs.coincall.com/}
\end{frame}

from coincall_client import signed_get   # from Week 1

def l2_top_of_book(symbol, key, secret):
    ob = signed_get(f"/open/option/order/orderbook/v1/{symbol}", {}, key, secret)
    data = ob["data"]                       # confirm field names in the docs
    bids = sorted(data["bids"], key=lambda x: -float(x[0]))
    asks = sorted(data["asks"], key=lambda x:  float(x[0]))
    best_bid, best_ask = float(bids[0][0]), float(asks[0][0])
    mid    = 0.5 * (best_bid + best_ask)
    spread = best_ask - best_bid
    return mid, spread, best_bid, best_ask

\begin{frame}{Glossary --- Key Terms}
\footnotesize\begin{description}
\item[\color{ncsured}Limit order book (LOB)] all resting limit orders organized by price level
\item[\color{ncsured}L1 / L2 / L3] top-of-book / aggregated depth / per-order; Coincall provides L2 only
\item[\color{ncsured}Mid-price] the average of the best bid and best ask
\item[\color{ncsured}Adverse selection] loss from trading against better-informed counterparties
\item[\color{ncsured}Effective spread] actual cost relative to mid (estimated by the Roll measure)
\item[\color{ncsured}Price impact] how far the mid moves in the direction of an incoming trade
\item[\color{ncsured}Order-flow toxicity] the degree to which incoming flow is informed
\end{description}
\end{frame}
